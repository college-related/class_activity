Basic Tags
<p>

P tag is used for paragrapgh
<button> Button tag is used to create a button.
<div> Div is a block level tag that takes the whole width of the viewport.
<span> Span is inline level tag and takes only the text-content area.
<i> I tag is also a inline level tag. It is also used to make text itallic.
<h1>...<h6> H1 to H6 tags are used to make heading tag. H1 is the largest, H4 is the normal and H6 is the smallest.
Linking tags
<a>

A tag is link tag and is used to create a hyperlink text.
<link>

LINK tag is used to link external files and outer sources like icon, font etc
<script>

Script tag is used to link external js file or make internal js codes
Text-Decorating tags
<u>

U tag is is used to underline the text within.
<b>/<strong>

These two tags are used to make the text within bold.
<sup>

SUP tag is used to make the text float up like a power.
<sub>

SUB tag is used to make the text float down.
<del>

Del tag is used to make the text have a line-through.
Table tags
<table>

TABLE tag is used to make a table.
<tr>

TR tag is used to make a table row.
<td>

TD tag is used to make a table column.
<th>

TH tag is used to make a table header.
<thead>

THEAD tag is used to make a table heading semantic.
<tbody>

TBODY tag is used to make a table body semantic.
<tfoot>

TFOOT tag is used to make a table footer semantic.
<caption>

CAPTION tag is used to make a caption for the table.
Form tags
<form>

FORM tag is used to make a form template.
<input>

INPUT tag is used to make a input field of different types.
<select>

SELECT tag used to make a template to have options.
Semantic tags
<header>

HEADER tag is used to make a template for head of webpage.
<nav>

NAV tag is used to make a outer layer template for navigation bar.
<main>

MAIN tag is used to make a template layer for the main part of the webpage.
<aside>

ASIDE tag is used to make a template for articles that are at the side.
<article>

ARTICLE tag is used to make a template for articles of the webpage like blog, post, etc.
<footer>

FOOTER tag is used to make a template for foot of the webpage.
<address>

ADDRESS tag is used to make a template layout for the address or contact info.
Layout tags
<html> HTML tag is the main tag of the document where all the tags go except for head.
<head> HEAD tag is used in HTML to store the datas data. It contains tags like title, meta, link, etc.
<title> TITLE tag is used to name the document in the bar at top of web browser.
<body> BODY tag is the tag that contains all the tag ans info that is displayed in the wenpage.
Media tags
<img> IMG tag is used to import a image file in the webpage.
<video> VIDEO tag is used to import a video file in the webpage.
<audio> AUDIO tag is used to import a audio file in the webpage.
<source> SOURCE tag is used to determine the path of the source file of video and audio tags.
